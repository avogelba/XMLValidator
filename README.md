# XMLValidator
Console application to validate XML against XSD

## Usage
`XMLValidator <file.xml> <shema.xsd>`

## Output
only Errors, if files are valid there is no output

## Exit codes
- 1 if there was an error
- 0 if all was ok

## .NET Version
4.5.2

## Libraries Used
none except .NET
